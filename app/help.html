Simple Image Placeholder Help

This application generates placeholder images based on input parameters provided via the query string.

Usage

To generate an image, make a GET request to the root URL (/) with the desired parameters in the query string.

Parameters

  • width (optional, default: 320): The width of the image in pixels. It must be a positive integer between 1 and the maximum width allowed (defined in the config.py file, default is 1920).
  • height (optional, default: 200): The height of the image in pixels. It must be a positive integer between 1 and the maximum height allowed (defined in the config.py file, default is 1080).
  • text (optional, default: "Hello, World!"): The text to be displayed on the image. It must be a string with a maximum length of 255 characters (defined in the config.py file).
  • font_size (optional, default: 30): The font size of the text. It must be a positive integer between 1 and the maximum font size allowed (defined in the config.py file, default is 100).
  • bg_color (optional, default: "grey"): The background color of the image. It can be a valid named color or a valid hex color code (3 or 6 digits, without the '#' character).
  • format (optional, default: "webp"): The format of the exported image. It can be one of the following: "jpeg", "png", "webp", or "svg".

Examples

  • Generate a PNG image with the text "Hello!" on a blue background, with a font size of 40, and dimensions of 800x600 pixels:

    http(s)://your.end.point/?width=800&height=600&text=Hello!&font_size=40&bg_color=blue&format=png

  • Generate a SVG image with the text "Welcome!" on a light grey background, with a font size of 20, and dimensions of 400x300 pixels:

    http(s)://your.end.point/?width=400&height=300&text=Welcome!&font_size=20&bg_color=F00&format=svg

Configuration

The application's configuration is defined in the config.py file. You can customize various parameters to suit your needs. For more information, please refer to the README.md file.
